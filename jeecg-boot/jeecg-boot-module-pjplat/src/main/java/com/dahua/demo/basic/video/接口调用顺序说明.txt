详细业务场景见 http://open.dahuatech.com/ 接口文档中业务介绍--》视屏基础--》视屏取流中的内容

1，首先调用【分级获取组织】OrgTree获取到当前环境的的所有组织和归属于该组织的通道信息
2，选择想要查看视屏相关内容的通道，使用其channelCode
    2.1通过【获取实时监视URI】RealMonitor获取预览取流URL，使用获取到的URL实现实时预览；
    2.2通过【录像查询】Record获取通道录像信息列表，
    2.3也可以通过【获取回放URI】Playback获取回放时间段内的录像文件URL，使用获取到的URL实现视频录像回放。
